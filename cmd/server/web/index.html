Usage

Point any OpenAI-compatible client at this grid:

Python:

from openai import OpenAI
client = OpenAI(
    base_url="",
    api_key="local-grid"
)
client.chat.completions.create(
    model="your-model",
    messages=[{"role": "user", "content": "hi"}]
)

About Beagrid

Beagrid is a free, open-source inference grid that pools your local AI engines (Ollama, vLLM, LM Studio, MLX, llama.cpp) behind one OpenAI-compatible endpoint. Requests are routed to the least-loaded engine automatically.

No accounts. No cloud. No relay. Just your hardware, connected.
